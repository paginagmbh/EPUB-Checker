README
++++++++++++++++++

pagina EPUB-Checker is a Java program provided to you as a 'runnable JAR' file.
This means, you can start the program by double-clicking the file "paginaEPUBChecker.jar".

Please make sure you installed a Java Runtime Environment.
Either the Oracle (Sun) JRE or OpenJDK packages.

If the JAR-File is being opened in Archiver program instead of showing the GUI, please check the following:
1. make sure you installed a Java Runtime Environment
2. right click the JAR file and select "Properties"
	1. Open the tab "Permissions" and make sure the checkbox at "Execute" is checked
	2. Open the tab "Open With" and make sure one of "OpenJDK Java 6/7 Runtime" or "Sun Java 6 Runtime" is selected
	3. Close the properties window and try again
